Octocat

Instructions

  1. Fork the instructor's repo named octocat
  2. Download an octocat image for each team member from the Internet. Size the image to 400 x 400 if you are familiar with image editing tools such as Photoshop or Picasa etc. If you are not familar with any of these tools then get help from another student or the instructor.
  3. Save the image in the images folder.
  4. Edit the octocat.html file under the heading with your name so that the file will show your octocat image and include a name for the octocat. If you are not familiar with HTML, use the " Classic" example as a guide.
  5. Delete the instructions from the file before committing and merging
  6. Delete the other student's names, however, keep your team member's names
